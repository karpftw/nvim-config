When making changes to the config files and you need to commit them:
- cd ~/./config/nvim
- git add .
- git commit -m "Comments go here"
- git push

On any local system:
- cd ~/.config/nvim
- git pull

Clone the repository:
- git clone https://github.com/yourusername/nvim-config.git nvim
